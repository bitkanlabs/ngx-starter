# ngx-starter

```
ng new --routing --style=scss --prefix=<prefix> <project name>
cd <project name>
ng add @angular/pwa
ng g universal --client-project=<project name> --appId=<appId>
ng g app-shell --client-project=<project name> --universal-project
ng add @angular/material
```
